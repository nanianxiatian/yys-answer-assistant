# 阴阳师答题助手

一个基于Python和PaddleOCR的本地OCR答题助手，支持导入Excel题库，自动识别游戏题目并匹配答案。

## 功能特点

- **本地OCR识别**：使用PaddleOCR，无需API调用，完全本地运行
- **区域选择**：可视化选择游戏画面中的识别区域
- **Excel题库支持**：支持.xlsx格式题库，自动识别问题/答案列
- **智能模糊匹配**：使用多种算法匹配题目，返回最可能的答案
- **定时识别**：可设置识别间隔，自动监控题目变化
- **纯识别无操作**：只做文字识别，不做任何点击操作，安全可靠

## 安装依赖

```bash
pip install -r requirements.txt
```

**注意**：第一次运行时会自动下载PaddleOCR模型文件（约100MB），请确保网络连接。

## 题库存储

本工具使用 **SQLite 数据库** 存储题库，相比 Excel 有以下优势：

- **更快的读取速度** - SQLite 读取速度是 Excel 的 10-100 倍
- **更小的文件体积** - 相同数据量，SQLite 文件比 Excel 小 30-50%
- **更好的查询性能** - 支持索引，搜索更快
- **零配置** - 无需安装数据库软件，Python 内置支持

### 数据导入

题库数据从 **Excel 文件导入**，导入后自动保存到 SQLite 数据库：

1. **首次使用**：加载 Excel 题库文件，数据自动存入 SQLite
2. **增量加载**：多次加载 Excel，会自动合并到现有题库
3. **自动恢复**：程序启动时自动从 SQLite 加载题库

### 数据备份

使用界面的 **"导出题库"** 按钮，可以将 SQLite 中的题库导出为 Excel 备份。

### 迁移旧数据

如果有旧版的 Excel 题库，使用迁移工具：

```bash
python migrate_to_sqlite.py
# 或指定文件
python migrate_to_sqlite.py questions.xlsx
```

### 数据库文件

- **默认位置**: `question_bank.db`（程序目录下）
- **配置文件**: `bank_config.json`（记录已加载的文件列表）

## 使用方法

### 1. 准备题库

创建一个Excel文件（.xlsx），格式如下：

| 问题 | 答案 |
|------|------|
| 以下哪个不是阴阳师中的SSR式神？ | N卡 |
| 阴阳师中，姑获鸟的别称是什么？ | 鸟姐 |

列名会自动识别，支持"问题/题目/question"和"答案/answer/ans"等常见命名。

### 2. 运行程序

双击 `start.bat` 或运行：

```bash
python main.py
```

### 3. 使用步骤

1. **加载题库**：点击"浏览"选择Excel题库文件，然后点击"加载"
2. **选择区域**：点击"选择区域"，拖动鼠标框选游戏右上角的题目区域
3. **设置间隔**：调整识别间隔（建议2-3秒）
4. **开始识别**：点击"开始识别"，程序会定时识别题目并显示匹配结果
5. **查看结果**：在"匹配结果"区域查看识别文本和匹配到的答案

## 界面说明

- **题库设置**：选择并加载Excel题库
- **识别区域**：显示当前设置的截图区域坐标
- **识别设置**：设置识别间隔和显示结果数量
- **识别状态**：显示当前状态和上次识别内容
- **匹配结果**：显示识别文本和匹配到的题目答案

## 匹配算法

使用综合匹配算法，包括：
- 部分字符串匹配（partial_ratio）
- 词序无关匹配（token_sort_ratio）
- 序列相似度（SequenceMatcher）
- 关键字匹配（基于中文分词）

结果按匹配度排序，优先显示匹配度高的答案。

## 注意事项

1. **游戏窗口**：确保游戏窗口在屏幕可见区域，不要被其他窗口遮挡
2. **区域选择**：选择包含题目的清晰区域，避免包含过多无关内容
3. **识别间隔**：根据答题时间设置合适的间隔，建议2-3秒
4. **题库质量**：完整准确的题库能提高匹配成功率
5. **OCR准确率**：文字清晰度会影响识别效果，尽量保持游戏画面清晰
